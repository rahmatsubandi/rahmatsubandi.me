## Step Satu

### Langkah pertama saat ingin menggunakan Express Js adalah:

Buka website [Express.js](https://expressjs.com/)

Kemudian buka ke Terminal

ketikan:

```shell:terminal
npm install express --save
```

To run it, namely by:

```shell:terminal
node .\app.js
```

Dan untuk menggunakan server yang secara otomatis restart setiap kali ada perubahan, Anda perlu menginstal Nodemon

ketikan:

```shell:terminal
npm install -g nodemon
```

Fungsi -g adalah server global, sehingga Anda dapat mengaksesnya di mana saja

Kemudian untuk menjalankan server, ketik:

```shell:terminal
nodemon.cmd .\app.js
```

Tapi jangan lupa bahwa meskipun menggunakan nodemon kita harus me-refresh browser

Untuk menggunakan template ejs silahkan buka link ini [EJS](https://ejs.co/#install)

Cara menginstalnya:

```shell:terminal
npm install ejs
```
